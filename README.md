# 🔐 Password Security Checker

## 📌 Project Overview

Password Security Checker is a beginner-friendly Python project that checks whether a password is strong or weak based on basic security rules.

This project is being developed step by step to improve Python programming skills and understand password security concepts.

---

## ✨ Features (Version 1)

* Check minimum password length (8 characters)
* Check for uppercase letters
* Check for lowercase letters
* Check for numbers
* Check for special characters
* Display whether the password is **Strong** or **Weak**
* Show missing password requirements
